== Description ==

[Demo](https://tishonator.com/plugins/parallax-slider)

Responsive Horizontal Parallax Sliding Slider using Swiper.js.

= Shortcodes: =

Insert `[parallax-slider]` on a page to display a slider with data configured in Admin Panel -> Left Menu -> Parallax Slider Page. 

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `parallax-slider` folder to the to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to Admin Panel -> 'Parallax Slider' -> fill Slider fields
4. insert the shortcode [parallax-slider] to display Slider to any page or post
5. Enjoy!
